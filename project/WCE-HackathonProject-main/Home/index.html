Areal View of Customers

Currently there is few such application exist in market and mainly such kind of work manage by physically (men power). So in my solution we are trying to convert it from physically to digitalise so it will provide more accuracy and reliability to the user and work more efficiently as compared to other application. End user of our web app application will be the NG0’s and other will be the users they can access our web application according to their needs. In our application it has filteralization capability it will filter the food on two days basis for this we have created recommendation system which contains all the list of items with item quantity.

According to the problem above we have decided to make an web based application where there is different options are given to select if you are the consumer (user) or if you are the donor both have to do registration according to the choose domain. Basically our application is for NGO’s and other people can also use it. In our application there are different items are given according to variety in food. We have built a recommender system for the consumer based on the food items are frequently consumed and expiry. Therefore also helping the donor donate. Reducing food waste at retail, food service and household label can provide multifaceted benefits for both people and planet. There is a chatbot on the home page if user has any doubt how to order or how to donate or any query related to our website they can get answer there and we are also using filtration to check the food quality.
